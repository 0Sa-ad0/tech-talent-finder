# TechTalent Finder

A technical talent sourcing platform that helps recruiters find, evaluate, and manage candidates for technical roles.

## Features

- **Candidate Management**: Add, view, edit, and delete candidate profiles
- **AI-Powered Evaluation**: Evaluate candidates based on their skills and experience using Hugging Face
- **Interview Management**: Generate technical interview questions and record responses
- **Real-Time Updates**: Live dashboard updates using Socket.IO
- **Job Management**: Create and manage job postings
- **Email Communication**: Contact candidates directly from the platform

## Tech Stack

### Backend
- Node.js with Express.js
- MongoDB with Mongoose
- Hugging Face API for AI evaluations
- Socket.IO for real-time updates
- Nodemailer for email communication
- JWT for authentication

### Frontend
- React.js with React Router
- Context API for state management
- Axios for API requests
- Socket.IO client for real-time updates
- Tailwind CSS for styling
